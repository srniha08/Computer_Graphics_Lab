# Computer Graphics Lab

This repository contains Computer Graphics Lab programs written in the **C programming language**. The source code is written using **Code::Blocks** and executed in **Turbo C** through **DOSBox** using the **BGI (Borland Graphics Interface)** graphics library.

## Requirements

* Code::Blocks (for writing code)
* Turbo C
* DOSBox
* BGI Graphics Library

## How to Run

1. Write or edit the program in Code::Blocks.
2. Copy the `.c` file to the Turbo C directory.
3. Open DOSBox and mount the Turbo C folder.
4. Start Turbo C.
5. Open the program, compile it, and run it.
